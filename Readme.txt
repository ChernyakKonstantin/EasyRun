Выполнить пункты согласно скриншотам.
Переименовать скачанный файл в track.gpx
В файле settings.txt указать дату и время занятия (-7 часов) и его продолжительность(в минутах).
Запустить easy_run.py
Полученный файл res.gpx загрузить в Strava (скриншот 5)
Выбрать тип "Ходьба"
Сохранить

Дату можно ставить задним числом.
